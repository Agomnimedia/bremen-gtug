  • Providing screenshots

Features for the developer

Completely new APIs

  • Social API - making it even more easier to
    integrate contact data with your app
  • Calendar API - no more using undocumented
    APIs and databases to integrate your app
    with build-in calendar
  • Voicemail API, AndroidBeam, Sharing Widget

Features for the developer

New media capabilities

  • Camera - easy use of filters,
    snapshots while shooting video, face-detection
  • Media Effects - high-performance transformation filters images and videos
  • New media codecs and containers - (WebP, VP8, HTTP Live streaming, AAC...)

Features for the developer

New UI capabilities

  • GridLayout - performance improvements by flatter view hierarchies, designed for drag-and-drop design tools
  • TextureView - OpenGL rendering handled just as a normal view object(improvement of SurfaceView)
  • Hardware-accelerated 2D drawing - high-performance drawing capabilities (scale, rotate...)
