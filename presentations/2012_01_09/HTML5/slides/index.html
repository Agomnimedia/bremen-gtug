Einleitung

Hintergrund

  • Ersetzt HTML 4 und XHMTL 1 bzw. 2
  • Ablösung für Plugins wie Adobe Flash
  • Entwurf durch die Web Hypertext Application Technology Working Group (WHATWG)
  • Standarddisierung durch das World Wide Web Consortium (W3C)

Syntax-Beispiel

  • HTML 1-4: 
    <input type=text disabled>
<p>Absatz<br>2. zeile
  • XHTML 1: 
    <input type="text" disabled="disabled" />
<p>Absatz<br />2. zeile</p>
  • HTML5: 
    <input type='text' disabled="">
<p>Absatz<br>2. zeile</p>

HTML-Struktur

Header und Gesamtstruktur

<!DOCTYPE html>
<html lang="de">
  <head>
    <title>GTUG Präsentation</title>
    <meta charset="utf-8">
    <meta name="author" content="Jan Stalhut" />
    <link rel="stylesheet" href="styles.css" />
    <script src="slides.js">
  </head>
  <body>
    <!-- Content -->
  </body>
</html>

Inhalt-Strukturierung

  • section - Allgemeine thematische Gruppierung
  • article - Inhaltlich in sich geschlossener Abschnitt
  • aside - Hat nur indirekt mit Artikel bzw. Webseite zu tun
  • header - Kopfdaten eines Webseiten-Abschnitts
  • footer - Fußzeile eines Webseiten-Abschnitts
  • h1 bis h6 - Überschriften und Unterüberschriften
  • hgroup - Gruppierung von Überschriften
  • address - Kontakt zum Author eines Artikels
  • nav - Navigationsbereich mit Verlinkungen

Tabellarische Darstellung

Formulare

Tags für Formulare

  • form, input, select, option, textarea - bekannt von (X)HTML
  • fieldset - Gruppe von zusammengehörigen Formular-Elementen
  • legend - Beschriftung einer fieldset-Gruppe
  • label - Beschriftung des Formular-Elements mit der ID des Attributs for
  • button - Button mit weiteren HTML-Elementen als Beschriftung
  • optgroup - Gruppierung von Optionen eines select-Elements (optional mit label-Attribut)
  • dataset - Liste mit Vorschlägen (option) für Eingabefeld
  • Beispiel ...
<form>
  <fieldset>
    <legend>Beispiel-Eingabefelder</legend>
    <label for="exampleText">Eingabefeld:</label>
    <input type="text" id="exampleText">
    <br/>
    <label for="exampleSelect">Auswahlfeld:</label>
    <select id="exampleSelect">
      <optgroup label="Option-Group 1">
        <option>Option 1a</option>
        <option>Option 1b</option>
        <option>Option 1c</option>
      </optgroup>
      <option>Option 2</option>
      <option>Option 3</option>
      <optgroup label="Option-Group 4">
        <option>Option 4a</option>
        <option>Option 4b</option>
        <option>Option 4c</option>
      </optgroup>
    </select>
  </fieldset>
</form>
Beispiel-Eingabefelder

Eingabefelder-Typen - Web form 2.0 -

  • text, password - bekannt von (X)HTML
  • search - Suchfeld mit X-Button um Eingabe zu entfernen
  • email, url, tel - Kontaktdaten, passende Tastaturen am Smartphone
  • range - Schieberegler (optional mit min, max und step Attributen)
  • number - Numerische Eingaben mit Hilfe von Buttons zum in- und dekrementiern (auch min, max und step möglich)
  • date, week, month, time, datetime, datetime-local - Datum-/Uhrzeit­auswahl (auch min, max und step möglich)
  • color - Farb-Auswahl via Farbpalette
  • Beispiel ...
<label for="exampleInputRange">Bereich-Feld:</label>
<input id="exampleInputRange" type="range" />
<br/>
<label for="exampleInputNumber">Zahlen-Feld:</label>
<input id="exampleInputNumber" type="number" />
<br/>
<label for="exampleInputDate">Datums-Feld:</label>
<input id="exampleInputDate" type="date" />
<br/>
<label for="exampleInputColor">Farben-Feld:</label>
<input id="exampleInputColor" type="color" />

Eingabefelder-Attribute - Web form 2.0 -

  • disabled, value - bekannt von (X)HTML
  • placeholder - Angezeigter Text, solange keine Benutzereingabe erfolgte
  • autofocus - Feld mit Fokus beim Seitenaufruf
  • required - Zum Absenden des Formulars erforderliches Feld
  • list - verweis auf ein datalist-Element mit möglichen Eingabe-Werten
  • Beispiel ...
<form>
  <label for="exampleInputTextPlace">Textfeld:</label>
  <input id="exampleInputTextPlace" type="text"
      placeholder="Hier kann etwas rein." />
  <br/>
  <label for="exampleInputTextRequ">Pflichtfeld:</label>
  <input id="exampleInputTextRequ" type="text"
      required="required" />
  <br/>
  <label for="exampleDatalist">Eingabefeld mit
      Vorschlägen:</label>
  <input id="exampleDatalist" type="text"
      list="exampleData" />
  <datalist id="exampleData">
    <option>Android</option>
    <option>Apple</option>
    <option>Google</option>
  </datalist>
  <br/>
  <input type="submit" value="absenden">
</form>



Weitere Tags für Formulare

  • keygen - Generator für Schlüssel-Paare: privater Schlüssel wird im lokalen Keystore hinterlegt, öffentlicher Schlüssel an den Server gesendet
  • output - Feld für Ausgaben, mit JavaScript ansprechbar wie Eingabefeld
  • progress - Fortschrittsbalken
  • meter - Ergebnis-Balken anzeigen (mit min, max und title Attribut)
  • Beispiel ...

<input id="exampleInputRange" type="range"
    min="0" max="10" step="0.5" value="2.5"
    oninput="exampleOutputRange.value =
	    exampleInputRange.value" />
<output for="exampleInputRange"
    id="exampleOutputRange">2.5</output>

Text-Auszeichnungen

Alte HTML-Elemente (mit neuer Bedeutung)

  • i - Fachbegriffe, alternative Bezeichnung, Fremdwörter, Eigenname, ...
  • b - Schlüsselwörter, Produkt-Namen
  • em - Ausdrücklich zu betonende Wörter
  • strong - Wichtige Wörter, z. B. eine Warnung vor Gefahr
  • small - Nebensächlichkeiten, Anmerkungen, Kommentare
  • abbr - Abkürzung, Kürzel (optional mit Ausgeschriebener Form im title-Attribut)
  • sub, sup - Tief- bzw. Hochgestellter Text, tapischerweise innerhalb von var-Tags

Änderungen kennzeichnen

  • del - Entfernte Inhalte, mit Datumsangabe (Attribut datetime)
  • ins - Hinzugefügte Inhalte, mit Datumsangabe (Attribut datetime)

Neue Elemente zur Auszeichnung

  • s - Nicht mehr richtig bzw. relevant
  • q - Zitat, optional mit Quellenanagabe in cite-Attribut
  • cite - Author eines Zitats / Titel einer Arbeit (Buch, Film, Song, ...)
  • dfn - Begriff einer Begriffsdefinition
  • time - Datum/Uhrzeit, ggf. mit datetime-Attribut
  • code - Programmcode, XML oder ähnliches

Neue Elemente zur Auszeichnung - Fortsetzung

  • kbd - Benutzereingaben, wie etwa Alt+F4
  • samp - (Beispielhafte) Ausgaben eines Computer-Programms
  • var - Variablen in mathematischen Ausdrücken (auch im Kontext von Programmcode)
  • mark - Textmarkierung, etwa Suchbegriff in Suchergebnissen
  • wbr - Zeilenumbruch der bei Beadrf vom Browser eingefügt werden kann.

Element-Einbettung

Multimedia-Inhalte

  • img - Bild/Grafik, wie bekannt
  • video - (Beispielhafte) Ausgaben eines Computer-Programms
  • audio - Variablen in mathematischen Ausdrücken (auch im Kontext von Programmcode)
  • object - Java, Flash, etc. (Dinge die ein Browser-Plugin benötigen)
  • canvas - Fläche zur Generierung von Grafiken etwa mit JavaScript
  • map - Enthält <area-Tags, die Bereiche einer Grafik ancklickbar machen.
