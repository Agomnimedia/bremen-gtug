Chili Project und Hackerspace Bremen

  • Webbasiertes Projektmanagementsystem
  • Open Source
  • Bietet verschiedene Module: News, Wiki, Dateiensammlung, Projektarchiv, etc.
  • Hackerspace Bremen benutzt Chili Project zur Dokumentation
  • Verein befindet sich in Migrationsphase auf neuen Server

Chrome Extensions

  • Chrome Extensions sind Erweiterungen für den Browser Chrome
  • Kleine Programme, die die Funktionalität des Chrome Browsers modifizieren oder erweitern
  • Werden mit Webtechnologien, wie HTML, CSS und JavaScript geschrieben
  • Können aus dem Chrome Web Store installiert werden.
  • Dokumentation von Chrome Extensions
