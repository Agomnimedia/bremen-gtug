Geschichte

Entstehung

  • 1993 - Erster Vorschlag für Stylesheets
  • 1994 - Vorschlag für CSS von Håkon Wium Lie
  • 1996 - Publizierung von Level 1 durch W3C
  • 1998 - Veröffentlichung von Level 2 (CSS 2)
  • 2000 - Start der Entwicklung von Level 3 (CSS 3)
  • 2002 - Start der Arbeit an Level 2 Revsion 1 (CSS 2.1)
  • 2011 - Veröffentlichung von CSS 2.1

Selektoren (bisher)

  • CSS1:
    • HTML-Tag: p {}
    • Klasse mehrerer Elemente: div.cl {}
    • ID eines Elements: #first {}
  • CSS2: (Attribut-Abhängigkeit)
    • Attribut vorhanden: input[disabled] {}
    • Bestimmter Wert: input[type=email] {}
    • Bestimmtes Wort im Wert: div[class~=cl] {}
    • Bestimmter Wert-Präfix: span[lang|=de] {}

Selektoren (neu mit CSS3)

  • Erweiterte Attribut-Abhängigkeiten
    • Wert beginnt mit: div[id^=first_] {}
    • Wert endet mit: div[id$=_last] {}
    • Wert enthält: div[id*=_middle_] {}
  • Wird alles ab IE 7, FF 3, Safari 3, ... unterstützt!

Kombinatoren

  • CSS1:
    • Nachfahren eines Elements: #block a {}
  • CSS2:
    • Kind eines Elements: #block > p {}
    • Direkter Nachfolger eines Elements: #block + div {}
  • CSS3:
    • Nachfolger eines Elements: #block ~ div {}
  • Wird alles ab IE 7, FF 3, Safari 3, ... unterstützt!

Pseudoklassen (bisher)

  • CSS1: (für Link-Status)
    • Noch unbesucht: a:link {}
    • Schon besucht: a:visited {}
    • Gerade angeklickt: a:active {}
  • CSS2: (Dynamik)
    • Maus über Element: tr:hover {}
    • Tastatur-Fokus: input:focus {}

Pseudoklassen (neu mit CSS3)

  • Link-Dynamik:
    • Aktuelles Link-Ziel: :target {}
  • Für Formular-Elemente
    • Aktiviertes Element: input:enabled {}
    • Deaktiviertes Element: input:disabled {}
    • Ausgewähltes Element: input:checked {}
  • Allgemein
    • Negierung: a:not([href^="http://"]) {}
  • Wird alles ab IE 9, FF 3, Safari 3, ... unterstützt!

Strukturelle Pseudoklassen

  • Erstes Kind eines Elements (CSS2): li:first-child {}
    • Lestes Kind eines Elements: li:last-child {}
  • X. Kind eines Elements: li:nth-child(4) {}
    • X. Kind, von hinten: li:nth-last-child(6) {}
  • Einzel-Kind eines Elements: li:only-child {}
  • Erste Element eines Types: p:first-of-type {}
    • Prinzip wie oben: :last-of-type  :nth-of-type(2) ...
  • Leere Elemente: td:empty {}
  • Wird alles ab IE 9, FF 3.6, Safari 3, ... unterstützt!

Pseudoelemente

  • CSS1:
    • Erste Zeile: p:first-line {}
    • Erster Buchstabe: p:first-letter {}
  • CSS2:
    • Inhalt vor dem Element: abbr:befor {}
    • Inhalt nach dem Element: abbr:after {}
  • CSS3:
    • Obige Pseudoelemente jetzt mit zwei vorangestellten Doppelpunkten.
  • Wird alles ab IE 8, FF 3, Safari 3, ... unterstützt!

Eigenschaften (Basis, CSS 1)

  • Hintergrundfarbe/-bild/-position
  • Schriftfarbe/-art/-größe/-stil
  • Zeilen-/Buchstaben-/Wortabstend
  • Textausrichtung/Absatz-Einrückung
  • Listenformatierungen
  • Anzeige, Breite, Höhe, Innen-/Außenabstand
  • Rahmenbreite/-farbe/stil

Eigenschaften (Erweitert, CSS 2/2.1)

  • Tabellen: Rahmenabstand, Überschrift-Position, lHandhabung leerer Zellen
  • Absolute/Relative Positionierung, Unsichtbarkeit, Ebenen, maximale Breite/Höhe
  • Umrandungen (nicht, wie Rahmen, Teil des Box-Modells)
  • Maus-Cursor Darstellung anpassen
  • Handhabung zu großer Inhalte (z. B. ggf. Scrollbalken)
  • Sprachausgabe beeinflussen (nicht Firefox und IE)
  • Druckausgabe Beeinflussen (nicht Firefox)

Eigenschaften (Neues in CSS 3)

  • Hintergrundbilder (IE 9, Chrome 17, FF 8):
    • background (Mehrere kommaseparierten Werte)
    • background-origin (Box-Modell)
    • background-size (Größe einzelner Bilder)
  • Transparenz (IE 9, Chrome 17, FF 3.6):
    • opacity (Prozentuale Angabe)
  • Rahmen und Layout (IE 9, Chrome 10, FF 4):
    • border-radius (Runde Ecken)
    • box-sizing (Bezug der Größe: Außen oder Innen)

Eigenschaften (Mehr neues in CSS 3)

  • Schatten (IE 9, Chrome 10, FF 4):
    • box-shadow (Rechteckiger Schatten)
    • text-shadow (Schrift-Schatten)
  • Farbverläufe als Hintergrund (IE 10, andere mit Präfix):
    • Linear Gradient (waagerecht, senkrecht, diagonal)
    • Radial Gradient (kreisförmig)
  • Animationen (IE 10, andere mit Präfix):
    • transition (Eigenschaft, Dauer, Verzögerung)

Kompatiblität

  • Browserspezifische Präfixe:
    • -webkit- Chrome und Safari
    • -moz- Mozilla Firefox
    • -o- Opera
    • -ms- Internet Explorer
  • Untertützung für ältere Versionen des IE: PIE
    • Bibliothek für IE 6-9
    • Kombination aus IE-spezifischer htc-Datei und JavaScript
    • http://css3pie.com/

Weiterer IE-Workaround

  • HTML-Block für bestimmte IE-Versionen.
  • Einbindung einer alternativen IE-kompatiblen CSS-Datei
  • Beispielsweise für alle IE-Versionen < 9: 
    <!--[if lt IE 9]>
<link rel="stylesheet" type="text/css" href="/ie.css">
<![endif]-->
  • Für jeden anderen Browser nur ein Kommentar.
